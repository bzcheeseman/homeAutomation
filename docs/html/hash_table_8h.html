Adds an entry to the map. External use for adding a key, value pair to the map. Hashes the key, creates a new entry and adds it to the map.

Uses recursion to avoid collisions - if two keys hash to the same value but are not the same char *, the function increments the bin by 1 and reruns the insertion. This is repeated until an empty bin is found.

Parameters
